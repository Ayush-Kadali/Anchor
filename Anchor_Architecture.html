Request Flow / Servlet Mapping

Method URL Pattern Handler Action Result
GET / welcome-file Serve welcome file index.jsp
GET /login LoginServlet.doGet() Forward login.jsp
POST /login LoginServlet.doPost() Authenticate user Redirect dashboard.jsp
GET /register RegisterServlet.doGet() Forward register.jsp
POST /register RegisterServlet.doPost() Create user Redirect login.jsp
GET /logout LogoutServlet Invalidate session Redirect login.jsp
GET /api/devices DeviceScannerServlet Scan serial ports JSON response
GET /dbtest DbTestServlet JDBC connection test HTML test page
WS /terminal TerminalWebSocket WebSocket upgrade Echo server
⚠ AuthenticationFilter intercepts ALL requests matching pattern /* — checks for valid HttpSession before forwarding to servlets. Unprotected paths (login, register, static) are whitelisted.
